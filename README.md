# Research Wiki Engine

A lightweight, file-based publication engine powered by Python/Flask.
Designed for rapid publishing using markdown.

## Architecture
- **Engine:** Python Flask
- **Content:** Markdown files in `/content` folder
- **Style:** Minimalist Academic CSS
- **Deployment:** Coolify / Docker

## Usage
1. Add `.md` file to `content/`.
2. Push to GitHub.
3. Live instantly.
